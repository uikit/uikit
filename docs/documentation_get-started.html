Get started

Get familiar with the basic setup and structure of UIkit.

First of all you need to download UIkit. You can find the whole project and all source files on GitHub.

Download UIkit

File Structure

In the ZIP file you will find all UIkit CSS, JavaScript and font files ready to use for your project. The core framework of UIkit has almost no styling in order to keep it slim. Therefore we provide two addidional distributions with a gradient and an almost flat style. Each style comes as a single CSS file as well as a minified version. Some advanced components are also included, which are not part of the core framework and can be added seperately to your project.

<!-- UIkit with the basic style -->
uikit.css
uikit.min.css

<!-- UIkit with Gradient style -->
uikit.gradient.css
uikit.gradient.min.css

<!-- UIkit with Almost flat style -->
uikit.almost-flat.css
uikit.almost-flat.min.css

<!-- JavaScript and minified version -->
uikit.js
uikit.min.js

HTML Page Markup

First off, make sure you have a solid code editor, for example Sublime Text. You need to add the compiled and preferably minified UIkit CSS and JavaScript to the header of your HTML5 document. jQuery is required as well. And that's it!

Example

<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <link rel="stylesheet" href="uikit.min.css" />
        <script src="jquery.js"></script>
        <script src="uikit.min.js"></script>
    </head>
    <body>
    </body>
</html>

Once you have finished implementing UIkit into your webpage, take a look at the UIkit components and get an overview of what UIkit is offering.

UIkit autocomplete for Sublime

To work even more efficiently, we recommend that you install UIkit autocomplete via Package Control. This saves loads of time, as you won't have to look up and type all UIkit classes.
