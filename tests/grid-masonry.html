Javascript Options

Option Value Default Description
margin String uk-grid-margin This class is added to items that break into the next row, typically to create margin to the previous row.
first-column String uk-first-column This class is added to the first element in each row.
masonry Boolean false Enables or disables masonry layout for this grid.
translate Number 0 Parallax translation value. The value must be a positive integer. Falsy disables the parallax effect (default).
target String false Selector string for child elements to translate. Defaults to element's children.
