JavaScript Options

Option Value Default Description
reverse Boolean false Reverse direction.
velocity Number 50 The animation velocity (pixel/s).
start Number 0 The start of the animation in 0-100%.
pause Boolean false Pause scrolling on hover.
fade-size Number 50 The fade size. The value can be in vh, % and px. It supports basic mathematics operands + and -.
pause-velocity Number 10 The animation velocity while being paused (pixel/s).
