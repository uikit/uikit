Option Value Default Description

offset-top false Boolean Subtracts the element's offset top from its height.
offset-bottom false Boolean|Number|Pixel|selector Subtracts the element's immediately following sibling's height (true), the given percentage (Number), Pixel (px) value from its own height or the given element's height.
