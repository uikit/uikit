功能特性

UIkit 是一个全面集合了HTML、CSS、JS的综合组件
它使用非常简单，自定义化和扩展也极其方便。

LESS

UIkit采用LESS进行编写，这使得它的代码结构良好、可扩展性强、容易维护。

组件

组件采用一致且无冲突的命名约定，除此之外它还足够精简、使用响应式布局。

自定义

UIkit's very basic style can be extended with themes and is easy to customize to create your own look.

Responsive

With the mobile-first approach UIkit provides a consistent experience from phones and tablets to desktops.

Open Source

UIkit is an open source framework to make the web look good.
Create your own styles and discover the unlimited use of UIkit.

MIT License

UIkit is open source and MIT licensed. It is absolutely free of charge and you can use, copy, merge, publish and distribute the framework without any limitations.

Our Heros

A big thanks to LESS, jQuery, normalize.css and FontAwesome. UIkit wouldn't be possible without these awesome open source projects.

Active Development

UIkit is built by YOOtheme, using our extensive experience in creating Joomla and WordPress themes, to power the Warp theme framework.

Get involved

UIkit is hosted on GitHub and open for everyone to contribute.
Please give us some feedback and join the development!

Contribute

You want to help us and participate in the development or the documentation? Just fork UIkit on GitHub and send us a pull request.

Found a bug?

Open a new issue on GitHub. Please search for existing issues first and make sure to include all relevant information.

Share themes

You created a beautiful UIkit theme? We would love to share it with the community and add it to the UIkit customizer. Send us a pull request.
